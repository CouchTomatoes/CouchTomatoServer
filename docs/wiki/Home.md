# CouchTomato Wiki

CouchTomato is a Python 3 port and rebrand of [CouchPotato](https://github.com/CouchPotato/CouchPotatoServer),
the automatic NZB and torrent movie downloader. See the [main README](../../README.md) for a project
overview, supported providers/downloaders, and screenshots.

## Pages

- **[Installation](Installation.md)** — install CouchTomato on Windows, macOS, or Linux, from a
  packaged release or from source
- **[Migration](Migration.md)** — moving from upstream CouchPotato, or upgrading between CouchTomato
  versions
- **[FAQ / Troubleshooting](FAQ.md)** — known issues, dead external services, and common questions
- **[Architecture & Porting Status](Architecture-Overview.md)** — what's been ported to Python 3 so
  far, what's left, and where to find the detailed history

## Getting help

Open an [issue](https://github.com/CouchTomatoes/CouchTomatoServer/issues) if the FAQ doesn't cover
your problem — check [`TODO.md`](../../TODO.md) first in case it's already a known, tracked item.
