# Branding assets

- `social-preview.png` — 1280x640 card for GitHub's social preview (shown when the
  repo link is shared on Slack/Twitter/etc). GitHub doesn't expose this setting
  through the API, so it has to be uploaded manually: repo **Settings → General →
  Social preview → Edit → Upload an image**. Built from the same tomato icon used
  for the app's favicon (`couchpotato/static/images/icons/windows.png`), rendered
  via headless Chromium at the exact size GitHub recommends.
